# Mobile App

React Native mobile application with Expo and NativeWind (TailwindCSS).

## Features

- React Native with Expo
- TypeScript
- NativeWind (TailwindCSS for React Native)
- React Navigation
- ZegoCloud SDK for video calls
- Firebase Authentication
- Socket.io for real-time features

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy environment file:
```bash
cp .env.example .env.local
```

3. Start development server:
```bash
npm start
```

## Run on Device

### iOS
```bash
npm run ios
```

### Android
```bash
npm run android
```

### Web (for testing)
```bash
npm run web
```

## Testing

```bash
npm test
```
